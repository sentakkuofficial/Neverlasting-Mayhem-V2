# NEVERLASTING: GRAND SELECTION MAYHEM

> 600 contestants enter. Only one survives.

A fast-paced platform fighter inspired by **Gun Mayhem**, **Super Smash
Bros.**, and the world of **Neverlasting: Origins**.

Battle through the deadly **Grand Selection Tournament**, wield powerful
abilities, launch opponents off the stage, survive dynamic hazards, and
uncover the secrets behind the Owari No Nai.

------------------------------------------------------------------------

## 🎮 Play

Open the game file directly in your browser:

``` text
[neverlasting_grand_selection_even_more_levels.html](https://sentakkuofficial.github.io/Neverlasting-Mayhem-V2/)
```

No installation, build tools, or dependencies required.

------------------------------------------------------------------------

## 🌐 Deploy with GitHub Pages

1.  Create a new GitHub repository.
2.  Upload:
    -   `index.html`
    -   `README.md`
3.  Go to **Settings → Pages**.
4.  Under **Source**, select:
    -   **Deploy from a branch**
    -   Branch: `main`
    -   Folder: `/ (root)`

Your game will be live at:

``` text
https://yourusername.github.io/repository-name/
```

## 🌌 About Neverlasting

**Neverlasting: Origins** is an original fantasy universe set in the
world of **Owari No Nai**.

Every person possesses either:

-   **Kaichi** --- elemental affinities
-   **Daichi** --- unique powers

Follow Ichido's journey through survival, betrayal, and the Grand
Selection Tournament.
